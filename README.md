# DIS23-b
Data Intern Summer 2023 Team B

# らしさ

## Setup
```bash
$ git clone https://github.com/gochipon/DIS23-b
$ cd DIS23-b

# .env.exampleを参考に.envを作成
$ touch .env

$ sudo docker compose build
```

## Run
```bash
$ docker compose up
```

## development
```bash
$ docker compose up -d

# front
$ docker compose exec front /bin/bash

# api
$ docker compose exec api /bin/bash

# view logs
$ cat api/logfile.log
```
